Як встановити локалізацію

Спосіб №1

  1. Скачайте .zip архів з GitHub Release
  2. Витягніть файли у StarCitizen/USER/LOCAL
  3. Відкрийте гру і насолоджуйтеся!

Спосіб №2

  1. Скачайте .zip архів з GitHub Release
  2. Відкрийте консоль
  3. Наберіть команду unzip <ваш файл>
  4. Готово!
